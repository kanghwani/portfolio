  • 빌드별(1차/2차/최종) 업무분담표로 팀원 산출물·완료 기준 명확화, dev-log로 전 과정 문서화

molten-arena.unity완성 · 발표

Molten Arena 3인칭 웨이브 생존 · 로그라이트

부트캠프 1차 팀 프로젝트 · 2인 · 1개월 · 클라이언트 구현

Unity URP 기반 TPS를 데이터 설계부터 타격감까지 폭넓게 구현하고, 부트캠프 1개월 팀플을 완주·발표했습니다.

// 내가 만든 것
  • ScriptableObject로 무기·아이템 데이터층 설계 + 런타임 상태 분리 → 새 무기 추가 시 코드 0줄로 확장(OCP)
  • 타격감 시스템(히트스톱·VfxPool 풀링) · 이벤트 구독 인벤토리 · 통합 핫바 · 로그라이트 레벨업
  • 난이도 밸런싱 — 1판 클리어 ~3분 → ~6분으로 조정해 압박감 강화
/* 문제 → 해결 · 대표 사례 */
problem
일시정지(timeScale=0)에도 카메라가 계속 회전하고 커서가 안 풀림
constraint
회전 로직이 동료 코드에 있어 직접 수정 시 통합 충돌 위험
solution
CameraPauseGate를 별도 추가 — timeScale==0 감지해 회전 정지·커서 해제. 동료 코드 무수정
verify
일시정지·레벨업·결과화면 전부 카메라 고정·커서 해제 확인, 콘솔 에러 0
